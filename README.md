# Alfred SiYuan Search

Search SiYuan notes from Alfred with the keyword `sy`, then open the selected result directly in SiYuan.

## Features

- Search SiYuan notes from Alfred
- Open notes directly via `siyuan://blocks/...`
- Automatically detects available SiYuan port
- Works with Alfred 5
- No third-party Python dependencies required

## Requirements

- macOS
- [Alfred 5](https://www.alfredapp.com/) with Powerpack
- [SiYuan](https://github.com/siyuan-note/siyuan)

## Installation

### Option 1: Download Release
Download the latest `.alfredworkflow` file from the [Releases](../../releases) page and open it.

### Option 2: Import Manually
Import the exported `.alfredworkflow` into Alfred.

## Usage

1. Make sure SiYuan is running
2. Open Alfred
3. Type `sy` followed by a search keyword

Example:

```text
sy 算法
sy docker
sy project
